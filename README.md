# Medical Cost Prediction using Linear Regression 📊

## 📌 وصف المشروع
مشروع تعلم آلة يهدف إلى التنبؤ بتكاليف التأمين الصحي باستخدام خوارزمية الانحدار الخطي (Linear Regression)، من خلال تحليل البيانات وبناء نموذج تنبؤي يعتمد على عدة عوامل مؤثرة.

---

## 🧠 التقنيات المستخدمة
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 📊 خطوات العمل
- استيراد البيانات ومعالجتها  
- تحليل البيانات (EDA)  
- تحويل البيانات الفئوية إلى بيانات رقمية  
- تقسيم البيانات إلى تدريب واختبار  
- بناء نموذج Linear Regression  
- تدريب النموذج  
- التنبؤ بالقيم  
- تقييم النتائج وتحليل الأداء  

---

## 📈 النتائج
تم تدريب النموذج باستخدام Linear Regression وتم تقييمه من خلال مقارنة القيم الحقيقية بالقيم المتوقعة وتحليل توزيع الأخطاء.
